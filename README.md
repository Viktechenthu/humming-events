# Humming Events — Gwalior

Website for **Humming Events**, wedding & event planners in Tansen Nagar, Gwalior, Madhya Pradesh.

## Stack

- **Frontend:** Next.js (App Router) + Tailwind CSS v4, TypeScript
- **Backend:** Python (Vercel serverless function) at [`api/book.py`](api/book.py) — validates booking requests and returns a prefilled WhatsApp link
- **Hosting:** Vercel

## Sections

Hero · About · Services · Gallery (real event photos) · How it works · Book an appointment (form + WhatsApp/call/email/map links)

## Develop

```bash
npm install
npm run dev
```

## Deploy

```bash
vercel --prod
```
